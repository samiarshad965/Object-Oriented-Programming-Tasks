# SkyLink Airways: Airline Reservation System

## Project Overview
SkyLink Airways System is a console-based application developed in C++ using Object-Oriented Programming concepts. The system manages airline flights, passengers, ticket bookings, cancellations, and records through a menu-driven interface.

## Objectives
* Apply Object-Oriented Programming concepts.
* Manage flight and passenger information.
* Book and cancel tickets.
* Demonstrate inheritance, polymorphism, encapsulation, and abstraction.
* Implement a user-friendly console application.

## OOP Concepts Used:

### Encapsulation
Data members are protected and accessed through member functions.

### Inheritance
Flight Hierarchy:

* Flight
* DomesticFlight
* InternationalFlight
* CharterFlight

Passenger Hierarchy:

* Passenger
* EconomyPassenger
* BusinessPassenger
* FirstClassPassenger

### Polymorphism
Virtual functions are used for displaying details and calculating refunds.

### Abstraction
Base classes are used to represent common flight and passenger behavior.

## Classes Used
