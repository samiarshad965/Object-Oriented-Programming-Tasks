# Airline Reservation and Flight Management System

## Project Description
This project is an Airline Reservation and Flight Management System developed using C++ and Object-Oriented Programming concepts. The system manages flights, passengers, bookings, cancellations, and flight records.

## Features
- Add, remove, search, and list flights
- Register and manage passengers
- Book available seats
- Cancel tickets
- View passenger booking history
- Track flight occupancy and revenue

## Technologies Used
- Programming Language: C++
- Concepts: OOP (Classes, Objects, Encapsulation, Inheritance, File Handling)
- IDE: Dev-C++

## How to Run
1. Open the project in Dev-C++.
2. Compile all `.cpp` files.
3. Run `main.cpp`.
4. Use the menu options to perform operations.

## Project Output
The system displays flight details, passenger information, booking confirmation, cancellation status, revenue reports, and occupancy details.
